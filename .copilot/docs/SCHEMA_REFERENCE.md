# Authoritative Schema Reference

| Schema Name                | Required Fields                                                                                                                                                                                                                                                               | Optional Fields                                                                                                                                                                                                                                                           | Key Constraints                                                                                                                                                                                                                                                                                                                                                                                                                           | Special Handling                                                                                                                                                        |
| -------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `agent.schema.json`        | description — string with minimum length 1; schema omits dedicated description text                                                                                                                                                                                           | name — string minLength 1; argument-hint — string minLength 1; tools — array of strings with each entry minLength 1; model — string minLength 1; infer — boolean; target — string; mcp-servers — object with schema unspecified; handoffs — array with schema unspecified | Object permits additional properties; arrays must not contain empty strings; unconstrained object and array fields require manual validation                                                                                                                                                                                                                                                                                              | infer, target, mcp-servers, handoffs, and tools depend on runtime support for inference routing, MCP configuration, handoff orchestration, and registered tool bindings |
| `collection.schema.json`   | id — string of lowercase letters, digits, or hyphen, length 1–50, description "Unique identifier for the collection"; name — string length 1–100 with description text; description — string length 1–500 with description text; items — array containing 1–50 unique objects | tags — array of up to 10 unique lowercase or hyphen strings each length 1–30 with description text; display — object offering optional ordering, show_badge, featured with description text                                                                               | Root object forbids extra keys; each item object forbids extra keys and must include a repository-relative path matching `skills/<name>/SKILL.md` or `<type>/<name>.<ext>` (type is prompts, instructions, or agents) plus a kind drawn from prompt, instruction, agent, or skill; usage optional string; display.ordering accepts manual or alpha with default alpha; display.show_badge defaults false; display.featured defaults false | Preserve repository-relative paths and approved kind values; maintain uniqueness for items and tags; honor display defaults when fields are omitted                     |
| `instructions.schema.json` | description — string minLength 1; applyTo — string minLength 1                                                                                                                                                                                                                | name — string minLength 1                                                                                                                                                                                                                                                 | Object allows additional properties; all defined strings must be non-empty                                                                                                                                                                                                                                                                                                                                                                | applyTo must map to the intended Copilot surface; additional keys require downstream validation                                                                         |
| `prompt.schema.json`       | description — string minLength 1; agent — string minLength 1 with default value "agent"                                                                                                                                                                                       | name — string minLength 1; argument-hint — string minLength 1; model — string minLength 1; tools — array of strings with each entry minLength 1                                                                                                                           | Object allows additional properties; tools array must not include empty strings                                                                                                                                                                                                                                                                                                                                                           | agent value must match a registered agent identifier; tools list must reference available runtime tooling                                                               |
| `skill.schema.json`        | name — string comprising lowercase letters, digits, or hyphen, length 1–64; description — string length 10–1024                                                                                                                                                               | _(No optional fields defined beyond additional properties)_                                                                                                                                                                                                               | Object allows additional properties; name constraint enforces lowercase slug; description constraint enforces substantive narrative length                                                                                                                                                                                                                                                                                                | Treat any additional properties deliberately; name doubles as slug identifier                                                                                           |
